AI 装修互动全景生成系统

从户型文件到可互动的装修效果展示

支持图片、PDF、DXF 户型输入，自动识别房间，选择丰富装修风格，上传参考图，通过 AI 对话调整设计方向，并生成房间互动展示。

查看源码 部署说明
上传图纸 -> 自动识别 -> 风格预览 -> 批量生成 -> 复核发布 Next.js + Prisma + SQLite

多格式户型输入

图片建议 2000 x 1500 以上；PDF 优先矢量图；DXF 需二维平面图并保留房间文字。

风格对话调整

内置现代简约、奶油风、新中式、意式极简等 16 种预设，并支持参考图和对话微调。

服务端部署

完整后台依赖 API、SQLite、上传目录和图像模型密钥，推荐部署在腾讯云轻量服务器。
